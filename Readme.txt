1.tecLogin.html: 教师注册登录页面

3.authenticationEdit.html： 教师编辑注册认证信息页面
4.authenticationInfor.html: 教师查看认证信息页面

5.personnelManage.html:   班级人员管理页面，可查询学生，可剔除学生
6.stuInfor.html:  查看学生个人信息
7.stuList.html: 所有学生列表，可查看该教师所有学生。

8.estabClass.html: 创建班级
9.classList.html:  班级列表，该老师所教授的所有班级，可以再创建班级，可以查看，编辑班级信息，
		   可以批改该班级作文。
10.classInfor.html: 查看班级信息。
11.applicatList.html:申请列表，教师查看学生申请加入该班级的信息列表

----------------------------------------------------------------------------

12.settingAssignment.html:  发布作业
13.assignmentInfor.html：  查看作业信息
14.articleList.html: 该班级作文列表
15.waitCorrection.html：待批改作文
15.articleMarking.html: 作文待批改。
16.articleScore.html: 查看作文分数
